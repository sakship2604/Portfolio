Game Arcade Preview App

Why this project?

This project is my portfolio project mockup and prototype. I designed this project for Google certificate UX coursework. The project was chosen from Sharpen. It is a game preview application for a local arcades. It wants to allow users to have a look at games at-home book the games in advance and avail offers which might help arcade to get more customers to their business.

Research

First, I got fimilarised with the gaming industry and local arcades. I visited two local arcades as well in Vancouver. The aim was to understand the problem statement and product goal statement the beginning.

Research Objective

I collected quantitative as well as qualitative data to find out target audience and understand the user journey.
I used surveys as well as interviews of the employees at arcade and regular arcade visitors.
This helped me create a persona, journey map and story board.

Design Branding

I create a initial mind-map to collect some related words to the ideate the design.

Atomic Design & Typography

I wanted to add some color pop. I wanted the color to be fun, funky and attractive.
I chose the font from google fonts called Poppins.

Wireframes

First, I designed a low-fidility wireframe on paper that gave me an intial idea of placing the elements on the screen.

Then, I recreated the same design on Figma which turned out to be high-fidility wireframe with some imrprovements.

Mockups

Next based on all the data collected I created the mockups, a full version with icons and accessiblity features added to it. The prototype flow was designed based on the user story and journey maps.

Next Phase...

The testing phase is underway. The accessibility features can be improved in the app as the next step.
